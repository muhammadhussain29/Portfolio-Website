## Muhammad Hussain – Portfolio

This repository contains the source code for my personal developer portfolio, built with Next.js (App Router) and TypeScript.

### Running the project

- **Install dependencies**: `npm install`
- **Start development server**: `npm run dev`
- **Build for production**: `npm run build`
- **Preview production build**: `npm run start`

### Technologies

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS (or other CSS framework used in this project)

### Project structure

- `app/` – application routes and pages
- `components/` – shared UI components
- `public/` – static assets
